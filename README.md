# TallerPython

En este repositorio encontrará los 9 archivos los cuales contienen el taller sobre python terminado en su totalidad, estos 9 archivos están caracterizados por:

1. El primer archivo realiza una útil enseñanza sobre el manejo de cadenas en python.
2. El segundo archivo realiza una útil enseñanza sobre el manejo de túplas en python.
3. El tercer archivo realiza una útil enseñanza sobre el manejo de listas en python.
